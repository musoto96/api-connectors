guild.edit

URL

https://www.bitmex.com/api/v1/guild

HTTP Method

PUT

Response Type

Guild

Parameters

  • Form: name String

    Name of the guild, must be unique, must be at least 5 characters

  • Form: emoji String

    Emoji name.

  • Form: potDistributionPercent Double

    How much of the pot should be distributed to the guild members, must be between 0 and 100

  • Form: potDistributionType String

    How the pot should be distributed to the guild members, must be one of the following: ROLL_OVER, TOP_3, TOP_5, TOP_10, VOLUME_PERCENTAGE, TOP_3_BY_ADV,TOP_5_BY_ADV,TOP_10_BY_ADV, RANDOM

  • Form: potTraderId Double

    User ID of the guild member with order write permission for the pot

  • Form: description String

    Guild description, can be used to explain the guild to other users.

  • Form: twitter String

    Guild twitter handle.

  • Form: discord String

    Guild discord link.

  • Form: imgUrl String

    URL for the profile image of the guild, is used by clients to add some color to the guild, if no image is provided, a default image is used

  • Form: isPrivate Boolean

    Guild privacy status
